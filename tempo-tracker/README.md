# Gig Tempo

## Overview

Gig Tempo is a React application designed for musicians who need quick access to song tempos during rehearsals and live performances.

Users can create a library of songs by entering a song title and selecting a beats-per-minute (BPM) value. Songs are displayed in a list where they can be selected, played, stopped, or removed.

When a song is selected, the application generates a metronome click at the song's stored BPM and provides a visual beat indicator to help musicians stay on tempo.

## Features

* Add songs to a tempo library
* Enter a song title
* Select a BPM value from a dropdown menu
* Save songs to a list
* Display all saved songs
* Delete songs from the list
* Select a song from the list
* Start tempo playback
* Stop tempo playback
* Display a visual beat indicator

## Technology

* React
* Vite
* JavaScript
* CSS

## Project Goal

The purpose of this project is to create a simple and reliable tempo management tool for live musicians while providing experience with React development, GitHub collaboration, and team-based software development workflows.
