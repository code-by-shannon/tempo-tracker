# Tempo Tracker

Tempo Tracker is a React application designed to help musicians organize songs and quickly start a metronome at the correct tempo during rehearsals and live performances.

## Features

* Add songs with a title and BPM (beats per minute)
* Store songs in a MySQL database
* Display a saved song list
* Start the metronome at the selected song's tempo
* Delete songs from the list

## Built With

* React
* Vite
* PHP
* MySQL
* Web Audio API

## Future Plans

* User accounts and authentication
* Multiple setlists
* Search and filter songs
* Mobile-friendly interface
* Visual metronome animation
* Volume control and additional metronome sounds

## Purpose

This project was built to practice full-stack web development while creating a tool that solves a real problem for working musicians.
